# A股量化交易系统

## 项目简介

本项目是一个完整的A股量化交易系统，整合了**8因子选股策略**和**位置判断策略**，包含数据获取、因子计算、回测引擎、参数优化等完整模块。

### 核心特点

- **数据源：AKShare（完全免费，无需注册，无需token）**
- **可视化界面：Streamlit Web应用**
- **8因子选股**：价值(EP) + 成长(增速) + 反转 + 低换手 + 低波动 + 质量(ROE) + 利润质量
- **位置判断策略**：低位看基本面/消息面，高位看趋势/量价
- **完整回测引擎**：模拟真实交易成本（佣金/印花税/滑点）、T+1、涨跌停限制
- **每日操作报告**：详细的建仓/调仓/清仓记录（真实股票名称和价格）
- **止损止盈系统**：固定止损 + 移动止盈
- **自定义股票池**：支持预设和自定义股票池

## 快速开始

### 1. 安装依赖

```bash
cd quant_strategy
pip install -r requirements.txt
```

### 2. 运行可视化界面

```bash
# 启动Streamlit应用
streamlit run app.py
```

浏览器会自动打开 http://localhost:8501

### 3. 使用步骤

1. 在左侧边栏选择股票池（预设或自定义）
2. 设置回测区间
3. 选择策略
4. 点击「运行回测」按钮
5. 查看结果和每日操作明细

## 策略说明

### 8因子选股策略

| 因子 | 权重 | 类型 | 说明 |
|------|------|------|------|
| EP | 15% | 价值 | 盈利收益率 = 1/PE |
| 净利润增速 | 15% | 成长 | 同比增速 |
| 营收增速 | 10% | 成长 | 同比增速 |
| 短期反转 | 15% | 反转 | -20日收益 |
| 低换手率 | 10% | 流动性 | -换手率 |
| 低波动率 | 10% | 风险 | -波动率 |
| ROE | 10% | 质量 | 净资产收益率 |
| 低应计比率 | 5% | 质量 | 利润质量 |

### 位置判断策略

**低位（价格低于均线 + 近期下跌）：**
- 看基本面变化（业绩拐点、毛利率提升）
- 看消息面催化（政策利好、重大订单）
- 低位本身就是买入理由

**高位（价格高于均线 + 近期上涨）：**
- 看均线系统（MA5>MA10>MA20>MA60）
- 看量价配合（放量上涨健康）
